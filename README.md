Revert GitHub contributions and streak. A browser extension.
- [x] Total Contributions
- [x] Longest Streak
- [x] Current Streak
- [x] Streaks no more limited to one year
- [x] New Github Design compatible
- [x] Compatible with the new dark theme

## Browser support
